# Static mirror and citation crumbs

Neutral notes about files that exist purely so mirrors of this repo behave politely on the web.

**`CITATION.cff`**  
Machine-readable bib-style stub for catalogs that harvest software citations.

**`llms.txt`**  
Pointer list toward the Markdown sources; nothing fancier than a table of contents for crawlers that look for it.

**`docs/index.html`**, **`docs/sitemap.xml`**, **`docs/robots.txt`**, **`docs/.nojekyll`**  
Fed to GitHub Pages from the `/docs` folder if Pages is enabled—plain HTML recap plus sitemap stubs so indexing tools have something boring to fetch.

Nothing here is outreach advice; flip toggles because you personally want a static echo of the readme, not because growth manuals told you to.
