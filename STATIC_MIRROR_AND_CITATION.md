# Static mirror and citation crumbs

Boring appendix about ancillary files tossed in so mirrored copies behave politely online.

**`CITATION.cff`** behaves like bib metadata for catalogs that munch software stubs.

**`llms.txt`** points at Markdown sources nothing fancier than a lazy table of contents.

**`docs/index.html`**, **`docs/sitemap.xml`**, **`docs/robots.txt`**, **`docs/.nojekyll`** feed GitHub Pages if you flip `/docs`. Same words as the readme, uglier MIME type.

Nobody needs a playbook for growth hacking here. Toggle Pages because duplication amuses you, not because hustle culture yelled first.
